1. Introduction to Control Statements

Control statements are used to alter the flow of execution based on certain conditions or repeat a block of code multiple times.

The main types of control statements are:

  • Conditional Statements: `if`, `if-else`, `switch`
  • Looping Statements: `for`, `while`, `do-while`

2. Conditional Statements

if

The `if` statement allows you to execute a block of code if a specified condition is true:


#include <stdio.h>

int main() {
    int num = 10;

    if (num > 0) {
        printf("The number is positive.\n");
    }

    return 0;
}

if-else

The `if-else` statement executes one block of code if the condition is true and another block if it is false:


#include <stdio.h>

int main() {
    int num = -5;

    if (num >= 0) {
        printf("The number is non-negative.\n");
    } else {
        printf("The number is negative.\n");
    }

    return 0;
}

else-if

The `else-if` statement allows you to test multiple conditions:


#include <stdio.h>

int main() {
    int num = 50;

    if (num > 100) {
        printf("Number is greater than 100.\n");
    } else if (num > 50) {
        printf("Number is greater than 50 but less than or equal to 100.\n");
    } else {
        printf("Number is 50 or less.\n");
    }

    return 0;
}

switch

The `switch` statement allows you to execute one block of code from multiple options based on the value of a variable:


#include <stdio.h>

int main() {
    int day = 3;

    switch (day) {
        case 1:
            printf("Monday\n");
            break;
        case 2:
            printf("Tuesday\n");
            break;
        case 3:
            printf("Wednesday\n");
            break;
        default:
            printf("Invalid day\n");
            break;
    }

    return 0;
}

3. Looping Statements

for

The `for` loop is used to execute a block of code a specific number of times:


#include <stdio.h>

int main() {
    for (int i = 0; i < 5; i++) {
        printf("Iteration %d\n", i);
    }

    return 0;
}

while

The `while` loop continues to execute as long as the condition remains true:


#include <stdio.h>

int main() {
    int i = 0;
    while (i < 5) {
        printf("Iteration %d\n", i);
        i++;
    }

    return 0;
}

do-while

The `do-while` loop is similar to the `while` loop, but it guarantees that the block of code will be executed at least once:


#include <stdio.h>

int main() {
    int i = 0;
    do {
        printf("Iteration %d\n", i);
        i++;
    } while (i < 5);

    return 0;
}

4. Examples

Conditional Statements Example


#include <stdio.h>

int main() {
    int number = 7;

    if (number % 2 == 0) {
        printf("%d is even.\n", number);
    } else {
        printf("%d is odd.\n", number);
    }

    return 0;
}

Looping Statements Example


#include <stdio.h>

int main() {
    int i;
    for (i = 1; i <= 10; i++) {
        printf("%d\n", i);
    }

    return 0;
}
