1. Introduction to C Programming

History of C

C programming language was developed by Dennis Ritchie at Bell Labs in the early 1970s. It evolved from the B programming language and has become one of the most widely used programming languages for system and application development.

Features of C

  • Simple: C has a straightforward syntax, making it easy to learn.
  • Efficient: C provides low-level access to memory, which allows for efficient program execution.
  • Structured: C supports structured programming with functions and control statements.
  • Portability: C programs can run on various types of computer systems with minimal modifications.
  • Rich Library: C comes with a standard library of functions that facilitate common operations.

Structure of a C Program

A typical C program consists of the following components:

  • Preprocessor Directives: Instructions for the compiler to process before compilation (e.g., #include).
  • Main Function: The entry point of a C program, where execution begins.
  • Statements and Expressions: Instructions and operations performed by the program.
  • Comments: Text used to document the code, which is ignored by the compiler.

2. Basic Syntax and Structure

Comments

Comments are used to explain code and are not executed. They can be single-line or multi-line:


#include <stdio.h>

// This is a single-line comment

int main() {
    /* This is a 
       multi-line comment */
    printf("Hello, World!\n");
    return 0;
}

Data Types

C supports several data types, including:

  • int: Integer data type
  • float: Floating-point data type
  • double: Double precision floating-point data type
  • char: Character data type

int age = 25;
float height = 5.9;
double weight = 70.5;
char initial = 'A';

Variables

Variables are used to store data. They must be declared before use:


#include <stdio.h>

int main() {
    int num = 10;
    printf("Number: %d\n", num);
    return 0;
}

Operators

C includes various operators:

  • Arithmetic Operators: +, -, *, /, %
  • Relational Operators: ==, !=, >, <, >=, <=
  • Logical Operators: &&, ||, !
  • Assignment Operators: =, +=, -=, *=, /=

#include <stdio.h>

int main() {
    int a = 10, b = 5;
    printf("Sum: %d\n", a + b);
    printf("Difference: %d\n", a - b);
    return 0;
}

Control Statements

C provides control statements to manage the flow of execution:

  • if: Conditional statement
  • for: Looping statement
  • while: Looping statement
  • switch: Selection statement

#include <stdio.h>

int main() {
    int num = 10;

    if (num > 0) {
        printf("Positive number\n");
    } else {
        printf("Non-positive number\n");
    }

    for (int i = 0; i < 5; i++) {
        printf("%d\n", i);
    }
    
    return 0;
}

3. Examples

Simple C Programs

Here are some simple programs to get started:


#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Basic Input/Output

Programs can read input from the user and produce output:


#include <stdio.h>

int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);
    printf("You are %d years old.\n", age);
    return 0;
}
