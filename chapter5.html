1. Introduction to Functions

Functions in C are blocks of code that perform a specific task. Functions help in breaking down complex problems into smaller, manageable parts.

Functions provide several benefits:

  • Modularity: Functions help in breaking down complex programs into smaller, manageable functions.
  • Reusability: Functions can be reused across multiple programs or multiple times within the same program.
  • Maintainability: Functions make it easier to maintain and update code.

A function in C is declared with a function header, which specifies the return type, function name, and parameters (if any). The function body contains the code to be executed.

2. Function Types

Built-in Functions

C provides several built-in functions that perform common tasks. Examples include:

  • printf(): Prints formatted output to the console.
  • scanf(): Reads formatted input from the console.
  • strlen(): Returns the length of a string.

User-Defined Functions

You can also create your own functions to perform specific tasks:


#include <stdio.h>

// Function declaration
void greet();

int main() {
    greet(); // Function call
    return 0;
}

// Function definition
void greet() {
    printf("Hello, World!\n");
}

3. Function Parameters

Passing Arguments

Functions can accept arguments, which are values passed to the function:


#include <stdio.h>

// Function declaration
void printSum(int a, int b);

int main() {
    int x = 5, y = 10;
    printSum(x, y); // Function call
    return 0;
}

// Function definition
void printSum(int a, int b) {
    int sum = a + b;
    printf("Sum: %d\n", sum);
}

Return Values

Functions can also return values to the caller:


#include <stdio.h>

// Function declaration
int square(int num);

int main() {
    int x = 4;
    int result = square(x); // Function call
    printf("Square: %d\n", result);
    return 0;
}

// Function definition
int square(int num) {
    return num * num;
}

4. Examples

Example 1: Function with No Parameters


#include <stdio.h>

void printMessage() {
    printf("This is a function with no parameters.\n");
}

int main() {
    printMessage();
    return 0;
}

Example 2: Function with One Parameter


#include <stdio.h>

void printSquare(int num) {
    printf("Square: %d\n", num * num);
}

int main() {
    printSquare(7);
    return 0;
}

Example 3: Function with Multiple Parameters


#include <stdio.h>

void printProduct(int a, int b) {
    printf("Product: %d\n", a * b);
}

int main() {
    printProduct(4, 5);
    return 0;
}

Example 4: Function Returning a Value


#include <stdio.h>

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(3, 8);
    printf("Sum: %d\n", result);
    return 0;
}

Example 5: Function with Array as Parameter


#include <stdio.h>

void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}

int main() {
    int numbers[] = {1, 2, 3, 4, 5};
    printArray(numbers, 5);
    return 0;
}
