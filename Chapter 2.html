1. Introduction to Input/Output in C

Input and output operations are essential for interacting with the user and handling data. In C, these operations are performed using standard library functions.

The two main standard I/O streams are:

  • Standard Input (stdin): Input data from the keyboard or other input devices.
  • Standard Output (stdout): Output data to the screen or other output devices.

2. Basic Input/Output Functions

printf()

The printf() function is used to print formatted output to the console:


#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    printf("The value of PI is approximately %.2f\n", 3.14159);
    return 0;
}

scanf()

The scanf() function is used to read formatted input from the console:


#include <stdio.h>

int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);
    printf("You are %d years old.\n", age);
    return 0;
}

3. Formatted Input/Output

Formatting Output

The printf() function supports various format specifiers to control how data is displayed:

  • %d: Integer
  • %f: Floating-point number
  • %c: Character
  • %s: String

#include <stdio.h>

int main() {
    int num = 100;
    float pi = 3.14159;
    char letter = 'A';
    char name[] = "Alice";

    printf("Integer: %d\n", num);
    printf("Float: %.2f\n", pi);
    printf("Character: %c\n", letter);
    printf("String: %s\n", name);

    return 0;
}

Reading Formatted Input

The scanf() function also supports format specifiers for reading input:

  • %d: Integer input
  • %f: Floating-point input
  • %c: Character input
  • %s: String input

#include <stdio.h>

int main() {
    int num;
    float height;
    char letter;
    char name[50];

    printf("Enter an integer: ");
    scanf("%d", &num);
    printf("Enter your height: ");
    scanf("%f", &height);
    printf("Enter a character: ");
    scanf(" %c", &letter); // Note the space before %c
    printf("Enter your name: ");
    scanf("%s", name);

    printf("Integer: %d\n", num);
    printf("Height: %.2f\n", height);
    printf("Character: %c\n", letter);
    printf("Name: %s\n", name);

    return 0;
}
